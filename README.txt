Run "game.pyw" to start the game.

Left click to set cell to active/live.
Right click to kill cell.
Space moves game forward 1 cycle.
Enter makes the 'game' run.
Up and down arrows control the time between cycles.
C key to kill all cells.

If you come across any issues, contact me by posting them on the github issues page: https://github.com/AnWildDuck/Tower-Defence/issues
